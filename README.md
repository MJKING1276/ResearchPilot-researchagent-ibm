# ResearchPilot – AI-Powered Research Agent

## Overview

ResearchPilot is an AI-powered Research Agent developed as part of the IBM SkillsBuild & Edunet Foundation Internship Program. The project leverages IBM Watsonx and Granite foundation models to help users analyze, explore, and interact with research papers through natural language queries.

The system implements a Retrieval-Augmented Generation (RAG) architecture, allowing users to upload research papers, retrieve relevant information using semantic search, and generate contextual answers grounded in document content.

---

## Problem Statement

Researchers and students often spend significant time reading lengthy research papers, identifying key contributions, understanding methodologies, and comparing findings across multiple publications.

ResearchPilot addresses this challenge by providing an intelligent Research Agent that can:

* Summarize research papers
* Extract methodologies and key findings
* Answer research-related questions
* Compare information across papers
* Provide evidence-based responses from source documents

---

## Key Features

### Research Paper Upload

* Upload and process PDF research papers
* Automatic text extraction and indexing

### Semantic Search

* Retrieve relevant content based on meaning rather than keyword matching
* Vector similarity search using FAISS

### Question Answering

* Ask natural language questions about uploaded papers
* Receive contextual answers generated by IBM Granite models

### Evidence-Based Responses

* Display retrieved document chunks used to generate answers
* Improve transparency and explainability

### Multi-Paper Analysis

* Analyze information from multiple research papers
* Support literature exploration and comparison

          ↓

          ↓

          ↓
FAISS Vector Database
          ↓
User Query
          ↓
Similarity Search
          ↓
Relevant Context Retrieval
          ↓

## How It Works

1. User uploads research papers.
2. Text is extracted from PDFs.
3. Documents are split into chunks.
4. Embeddings are generated.
5. FAISS creates a searchable vector index.
6. User submits a question.
7. Relevant document chunks are retrieved.
8. Retrieved context is sent to IBM Granite.
9. Granite generates a contextual response.
10. Evidence and source information are displayed.

## Installation

### Clone Repository

```bash
git clone https://github.com/MJKING1276/ResearchPilot-researchagent-ibm.git
cd ResearchPilot-researchagent-ibm
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure Environment Variables

Create a `.env` file and add:

```env
IBM_API_KEY=YOUR_API_KEY
IBM_PROJECT_ID=YOUR_PROJECT_ID
IBM_URL=YOUR_WATSONX_URL
```

### Run Application

```bash
python app.py
```

Open the application in your browser.
